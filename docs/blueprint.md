# **App Name**: EcommPilot

## Core Features:

- Dashboard Overview: Provides a unified view of multi-channel e-commerce operations, including location-based inventory and growth analytics.
- Automated Recommendations: AI-powered recommendations based on real-time data to optimize inventory, ad spend, and profitability.
- Inventory Management: Manage and track inventory across multiple locations with real-time updates and stock level predictions. Determine the days to reorder using AI tool based on data from Google Sheets and existing stocks.
- Growth Analysis: Analyzes growth trends across different platforms to identify opportunities and areas for improvement.
- Daily P&L Tracking: Tracks daily profit and loss metrics to provide insights into financial performance.
- Google Sheets Integration: Syncs data from Google Sheets to provide real-time insights and analysis.
- Alert System: Notifies users of critical issues, such as low stock levels or underperforming ad campaigns, through a central alert box.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to convey trust and sophistication, drawing from the dashboard's theme.
- Background color: Light gray (#F8FAFC), a very low saturation of indigo, providing a clean and modern look.
- Accent color: Violet (#6366F1) to complement the indigo primary color, and to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif font, to ensure readability and a modern aesthetic.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and clear icons from Lucide to represent different data points and actions.
- Subtle animations for loading states and transitions to enhance user experience without being distracting.
